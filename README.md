# Activity_Booking
ProjectX - Application de Gestion des Activités Sociales
Description
ProjectX est une application web moderne développée pour simplifier la gestion des activités sociales au sein d'une organisation. Conçue avec une interface utilisateur intuitive, elle permet aux utilisateurs de s'inscrire, de consulter et de gérer différentes activités sociales proposées, tout en offrant aux administrateurs un contrôle total sur la gestion des inscriptions et des participants.

Fonctionnalités principales :
Inscription aux activités : Les utilisateurs peuvent parcourir les activités disponibles, s'inscrire, et suivre l'état de leur inscription.
Gestion des utilisateurs : Vue d'ensemble des utilisateurs inscrits à une activité, avec détails complets tels que le nombre de points cumulés et les informations personnelles.
Gestion des activités : Ajout, modification et suppression d'activités. Chaque activité peut avoir des catégories, des places limitées et une période d'inscription définie.
Liste d'attente : Si une activité atteint sa capacité maximale, les utilisateurs peuvent être placés en liste d'attente.
Récapitulatif des activités : Vue complète des activités passées et actuelles, des participants, et des revenus générés par activité.
Chatbot intégré : Les utilisateurs peuvent interagir avec un chatbot pour s'inscrire aux activités et poser des questions sur les événements.
Technologies utilisées :
Frontend : React pour une interface utilisateur dynamique et réactive.
Backend : Node.js et Express pour les API REST et la gestion des requêtes.
Base de données : MySQL pour la gestion des utilisateurs, des activités et des inscriptions.
Cloud : Hébergement sur Heroku avec JawsDB pour la base de données MySQL.
Gestion des utilisateurs : Utilisation de JWT pour l'authentification des utilisateurs et la gestion des sessions.
